# node-backend-template

This is a Node.js backend template.

## Setup

### Dev
- TypeScript
- ESLint (airbnb)
- Prettier

### Modules
- Node.js
- Express.js
- uuid
- date-fns

## Instructions

### Development
- Run ```yarn```;
- Run ```yarn dev:server```;
- Browse to ```http://localhost:3333/```.

### Build
- Run ```yarn build```;
- The bundle will be located in the ```./build``` folder.
